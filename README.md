# 📖 NEWSTART 성경 암송 플래시카드

성경 구절 암송을 돕기 위해 제작된 웹 기반 플래시카드 애플리케이션입니다.
빈칸 채우기(Fill-in-the-blanks) 방식을 통해 13과 말씀을 효과적으로 암기할 수 있도록 돕습니다.

## ✨ 주요 기능

* **빈칸 퀴즈:** 앞면에는 빈칸이 뚫린 문제, 뒷면에는 전체 말씀(정답)이 표시됩니다.
* **카드 뒤집기:** 카드를 클릭하거나 스페이스바를 눌러 앞/뒷면을 확인합니다.
* **네비게이션:** 화살표 버튼이나 키보드 방향키로 이전/다음 문제로 이동합니다.
* **반응형 디자인:** PC와 모바일(스마트폰) 환경 모두 지원합니다.

## 🛠 사용 기술

* HTML5
* CSS3
* Vanilla JavaScript

## 🚀 실행 방법

1.  이 저장소를 다운로드(Clone) 합니다.
2.  `index.html` 파일을 웹 브라우저(Chrome, Safari 등)에서 실행하면 바로 사용할 수 있습니다.
